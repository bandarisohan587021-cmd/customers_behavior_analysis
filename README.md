Data Analytics Project

Overview

This project demonstrates the complete data analytics workflow, starting from loading a dataset in Python to creating a final business dashboard and presentation. The project includes data exploration, data cleaning, SQL analysis, Power BI dashboard development, report writing, and presentation creation.

The aim of this project is to generate meaningful insights from the dataset and present them in a clear and professional format for decision-making.

Dataset

The dataset used in this project contains business-related information such as sales, customers, products, transactions, or performance metrics.

The dataset was used to:

* Understand data structure and quality
* Identify missing values and duplicates
* Perform exploratory data analysis
* Run SQL queries for deeper insights
* Create visual dashboards in Power BI

Tools Used

* Python – Data loading, cleaning, and exploratory data analysis
* Pandas – Data manipulation and analysis
* Matplotlib / Seaborn – Data visualization during EDA
* PostgreSQL / MySQL / SQL Server – SQL querying and database analysis
* Power BI – Dashboard creation and business visualization
* Gamma – Presentation creation
* Microsoft Word / Google Docs – Final report preparation

Project Steps

1. Data Loading

The dataset was loaded into Python using Pandas. Initial checks were performed to understand the number of rows, columns, data types, and overall structure of the dataset.

2. Exploratory Data Analysis

Exploratory Data Analysis was performed to understand patterns, trends, and relationships in the data.

Key activities included:

* Checking dataset shape and column information
* Identifying missing values
* Finding duplicate records
* Understanding numerical and categorical variables
* Creating basic charts and summaries
* Identifying outliers and unusual patterns

3. Data Cleaning

The dataset was cleaned to improve accuracy and reliability.

Cleaning steps included:

* Removing duplicate records
* Handling missing values
* Correcting data types
* Renaming columns for better readability
* Standardising text values
* Preparing the dataset for SQL and Power BI analysis

4. SQL Analysis

The cleaned dataset was imported into a database using PostgreSQL, MySQL, or SQL Server.

SQL queries were used to answer business questions such as:

* Total sales or revenue
* Top-performing products or categories
* Customer trends
* Monthly or yearly performance
* Region-wise or segment-wise analysis
* Key performance indicators

5. Power BI Dashboard

A Power BI dashboard was created to present the key insights visually.

The dashboard includes:

* KPI cards
* Sales or performance trends
* Category-wise analysis
* Region or customer segmentation
* Interactive filters and slicers
* Visual comparison of key metrics

Dashboard

The Power BI dashboard provides a clear overview of business performance and allows users to explore insights interactively.

Main dashboard features:

* Total revenue / sales overview
* Profit or performance indicators
* Trend analysis over time
* Product or category performance
* Customer or regional insights
* Interactive filtering options

Results and Insights

The analysis helped identify important patterns and business insights from the dataset.

Key findings include:

* Best-performing categories, products, or regions
* Trends in sales or performance over time
* Areas with high or low performance
* Customer behaviour patterns
* Opportunities for business improvement

These insights can support better decision-making and help businesses improve strategy, operations, and performance.

Report

A final report was prepared to explain the project process, analysis, findings, and recommendations. The report summarises the key insights from Python, SQL, and Power BI in a clear and structured way.

Presentation

A professional presentation was created using Gamma to explain the project in a simple and visually appealing format. The presentation includes the project overview, methodology, dashboard screenshots, key insights, and final recommendations.
